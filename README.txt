
Программа-симулятор для ночной игры на планете Пандора в рамках ролевой игры
"Полдень: 22 век" мастерской группы "Лестница в небо". Программа служит для
моделирования вирусологии и состоит из двух частей: симулятор, который
производит историю развития штамма бактерии жизни с запущенными в нее
вирусами, а также просмоторщик, который позволяет просматривать и изучать
по шагам развитие штамма. Вирусы попадали в модель из последовательностей
ДНК, которые привязывались ко всем живым организмам на Пандоре, в том числе
и к людям. Подробно игра в науку на Пандоре описана в статье:
http://fedoseev.net/Floran-RPG-Science.pdf

Программа требует библиотеку Shooes (http://shoesrb.com/) для запуска.
